# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-02

### Added
- Complete static analysis engine for Laravel applications
- Analyzers: Class, Model, Controller, Route, Database, Service, Test, Security
- Dependency graph builder with circular dependency detection
- Health score calculator with multiple metrics
- Test coverage analysis and missing test suggestions
- Security audit with vulnerability detection
- Documentation generators: JSON, Markdown, HTML
- Interactive dashboard with Vue.js 3 + Inertia.js + Tailwind CSS
- Mermaid.js dependency graphs and ER diagrams
- Chart.js test coverage visualizations
- Plugin architecture for custom analyzers and reporters
- Artisan commands for analysis, export, docs, security, and more
- Dark/light theme support in dashboard
- Caching system for incremental analysis

### Compatibility
- PHP 8.2, 8.3, 8.4
- Laravel 11.x, 12.x, 13.x
